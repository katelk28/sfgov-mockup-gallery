SF.gov · design hand-off

Content-page redesign — change log by batch

Each design direction, written as the concrete changes to make to today's live SF.gov content pages — so you can weigh the outcome against the effort.

How to read this. The base is today's live SF.gov site. Each batch is cumulative — it includes everything above it plus its own changes. So Batch 4 = Batch 1 + 2 + 3 + its deltas; the effort for a batch is that batch plus all the ones before it.
